# HTML Anything

你有一个专用的 HTML 生成工具 `html-anything_preview_html`，可以将生成的 HTML 推送到插件的可视化预览面板。

## 使用方式

1. 根据用户需求选择合适的模板
2. 使用 `html-anything_preview_html` 工具将生成的 HTML 推送到预览面板
3. 用户可以在插件的交互式页面中查看、编辑和导出 HTML

## 支持的内容类型

- **文章**: 杂志文章、博客文章、数字指南
- **演示**: 30+ 种演示文稿模板（Keynote、Pitch、Tech 分享等）
- **海报**: 杂志海报、营销海报、线框图
- **卡片**: 社交媒体卡片（Twitter、Reddit、Spotify）
- **原型**: 网页原型、SaaS 落地页、移动应用
- **报告**: 数据可视化、财务报告、仪表盘
- **文档**: 技术文档、邮件、运行手册、简历

## 设计约束

- 输出自包含单文件 HTML
- 中文优先 Noto Sans SC / Noto Serif SC
- 8px 基线网格
- 颜色对比度 ≥ 4.5
- 不使用占位符文本，必须使用用户真实数据
